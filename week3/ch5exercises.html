Chapter 5

Some examples are in the console and code comments.

  • Objects are self-contained sets of related values and functions.
  • Objects are used to keep related inforamtion and functionality together.
  • object properties that don't follow naming rules need to be quoted:
    • name: 'Superman',
    • 'real name': 'Clark Kent',
    • realName: 'Clark Kent'
  • Create an object literal as follows: const spiderman = {};
  • there is a shorthand method of creating objects, if the property key is the same as a variable name:
    • long: const ironMan = { name: name, realName: realName };
    • short: const ironMan = { name, realName };
  • Properties can be accessed two ways: superman.name -or- superman['name']
  • Computed property keys can be assigned with square brackets. ['catch' + 'Phrase'] becomes catchPhrase.
  • Property values can be experessions. {hero: bewitched ? false : true}
  • Methods can be called in two ways: superman.fly() -or- superman['fly']()
  • The 'in' operator can check whether an object has a property.
  • You can use for loops to cycle through all of the object's keys/values.
  • New properties are added by assigning a value to the property: superman.city = 'Metropolis'
  • Properties can be changed by updating the value: superman['real name'] = 'Kal-El';
  • Propertied are removed by using the delete operator: delete superman.fly
  • You can create nested objects, or objects that contain other objects.
  • Nested objects are accessed by referencing each property name in order: jla.wonderWoman.realName
  • Object literals can be passed as a parmeter to a function.
  • The keyword "this" refers to the object that it is within.
  • Putting functions that relate to each other inside an object is referred to as creating a namespace for them.
  • JSON: JavaScript Object Notation, property names must be double-quoted, functions are not permitted values.
    • .parse() takes a string of JSON and returns the Javascript object.
    • .stringify() takes the JavaScript object and returns a string of JSON data.
  • The Math object is built-in and has several properties for mathematical constants and methods.
  • The Date object contains information about dates and times and has several methods.
  • The RegExp object is a pattern used to search strings for matches.
