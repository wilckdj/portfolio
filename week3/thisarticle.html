Object Methods, "this"

Some examples are in the console and code comments.

  • Objects usually represent real world entities: user
  • Objects can be used to call functions: user.sayHi()
  • When creating objects, you can assign a function as a property of the object, a method.
  • The term for writing code using objects is object-oriented programming.
  • The keyword "this" references the object calling the method.
  • "this" allows for the code to work even if the object is copied to another variable.
  • "this" is not bound: it can be used in any function even if its not a method.
  • If a function is called without an object, "this" is undefined.
  • Arrow functions have no "this": if "this" is referenced from an arrow function, it references the outer normal function.
