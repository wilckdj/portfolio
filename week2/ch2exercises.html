Chapter 2

Some examples are in the console and code comments.

  • There are seven data types: String, Symbol, Number, Boolean, Undefined, Null, Object.
    • All Non-primitive data types return a type of object.
  • To declare variables, use 'const' or 'let'.
    • const is used for variables that are not going to be re-assigned to another value. Example: const name = "joseph", name will always be "joseph"
    • const for non-primitive data types is changable, such as changing the values in a const array.
    • let is used for variables that might be reassigned later. Example: let X = 5, X = 4
    • var should no longer used, but still works for backwards compatibility.
  • const and let variables only exist inside the block they are declared in.
  • Global Scope is when a variable is declared outside of a block, and is accesible from anywhere in the program.
  • Local Scope is when a variable is declared inside a block, {}.
  • The same variable name can have different values depending on the scope it is declared in.
    • You can declare a global variable from inside a block if you don't use let or const.
  • The best ways to convert data types is to use the corresponding function:
    • Number('100'); - Converts string to a number
    • String(100) - Converts number to a string
    • .toString() method - converts to number, the base can be specified.
  • Parsing numbers, instead of converting, can be done using parseINT() or parseFloat()
    • parseInt() is for integers and can also specify the base that you are converting.
    • parseFloat() is for floating point decimals.
  • Template Literals use the backtick ` to deliminate the string. `Hello` instead of 'Hello'
  • You can put javascript expression inside a string that will show the result instead of the script.
