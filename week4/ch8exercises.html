  • Forms generally submit information to be processed on the back end, but can be also processed on the front end.
  • Forms can be selected with JavaScript using form = document.forms[#], document.getElementsByTagname('form')[#], or document.forms.name.
    • The ideal method is: document.forms['name'];
  • The form elements can can be access/assigned through form.elements or form.name / form['name']
  • form.submit() will submit the form automatically.
  • forms can also be submitted manually by a button or input element with the submit type.
  • form.reset() will reset all the form controls back to initial values.
  • A button with a type of 'reset' will also reset the form.
  • form.action can be used to set the action attribut of a form so its sent to another URL for server processing.
  • Focus events occur when an element of a form is focused on.
  • Blur events occur when focus moves away from a form element.
  • Change events occur when the element is changed and the focus leaves the element.
  • Submit events occure when the form is submitted, usually to send the content to the server.
  • Javascript can intercept the form before it is sent using a submit event listener.
  • preventDefault() can stop the submission from being sent.
  • You can establish a default value for the input using input.value = "";
  • You can use focus and blur event listeners to clear and add a default value such as "Search Here".
  • An alternative is to use the placeholder attribute, but it doesn't act as a value for submission.
  • Common fom controls are: input, select, textarea, and button.
  • The input element includes attirbutes such as autofocus, placeholder, and maxlength.
  • There are multiple types of input fields:
    • 'text' is the default, for typing in short pieces of text
    • 'password' is like text but conceals the characters for secret information.
    • 'checkbox' is for true or false values.
    • 'radio' is for exclusive choices, where only one can be selected.
    • 'hidden' is for values that do not need to be displayed but should be submitted with the form, not secure.
    • 'file' is for uploading files.
    • others such as 'number', 'tel', 'color', 'date'
  • Select Drop-Down Lists are used for choosing options from a list of values.
    • The 'selected' attribute indicates which options value is used.
    • The 'selectedIndex' property returns the numeric index of the option selected.
    • the 'multiple' attribute is needed if more than one option will be selected.
  • 'textarea' elements are used to enter longer text, and can be over multiple lines.
  • There are multiple types of buttons: 'submit', 'reset', 'button', 'menu'
  • Form validation includes checking for certain values before allowing it the form to submit.
  • Submit buttons can have the 'disabled' attribute.
